About Calculator Hub

🎯 What is this?

Calculator Hub is a comprehensive web-based calculator application that provides multiple calculation tools in one place.

✨ Features

  • Mathematical Calculator: Basic and advanced math operations
  • Financial Calculator: Loans, discounts, ROI, VAT, compound interest
  • Health Calculator: BMI, BMR, water intake, TDEE
  • Unit Converter: Length, weight, temperature, speed, data
  • History: Track your calculations with SQLite storage

🛠️ Technology Stack

  • Backend: FastAPI (Python)
  • Frontend: Vanilla JavaScript, CSS3
  • Database: SQLite (file-based)
  • Design: Modern dark theme with animations
